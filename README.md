### (Re)Writing History with Git

#### Topics

1. Fixing the last commit

`commit --amend`

2. What is HEAD, and usage of ~

3. Removing commits

`reset (soft, hard)`

4. Displaying all interactions

`reflog`

5. Going back to a commit that was reset

`reset <commit_hash>`

6. Updating the commit message of a commit from anywhere on the logs

interactive rebase: `reword`

7. Updating changes from a commit from anywhere on the logs 

`commit --fixup` and interactive rebase: `fixup`

8. Pushing re-written history to remote

`push -f`
